Project Vision Documentation

Overview

This project aims to develop a smart city solution for managing urban traffic, utilizing real-time data to improve traffic management and enhance urban mobility. The project is built using Flask for the backend, Bootstrap 5 for the frontend, and integrates the HERE API for fetching real-time traffic data.

Quickstart

If you are interested in contributing to or building your own extension of our website, you are welcome to look at the installation and usage section in the Readme.md file in the Github Repository of our project.

Features

  • Real-time traffic data visualization using Leaflet maps.
  • Dynamic updates of traffic conditions, including jam factor and data availability.
  • User-friendly interface with Bootstrap 5.

Future Work

  • Implement automated city lights control.
  • Develop smart parking solutions.
  • Enhance data analytics for predictive traffic management.
